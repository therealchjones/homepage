The State on iTunes!

9/26/2006 04:26:00 PM

Holy crap, that bastion of mid-90's comedy The State is now available on iTunes! I've been a big fan for years, but the show is never re-run and a DVD has been “in the works” for a ridiculously long time now. The first season's availability in Apple's famous “near-DVD” quality means I'm out another ten bucks. With any luck, that'll make an actual DVD show up that much faster….

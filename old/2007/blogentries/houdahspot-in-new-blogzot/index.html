HoudahSpot in New BlogZot

6/17/2006 08:32:00 AM

macZOT! is at it again, offering to give away software if enough people blog about it. This time, it's HoudahSpot, an enhanced front end for the Spotlight desktop search engine in Mac OS X.

After trying it out for a bit, to be honest, I'm not sure most people will find it all that useful. I do pretty basic Spotlight searches, usually for one word in a file's name or description, which works pretty well in the built-in interface without a need to use something more advanced. Still, there are some benefits:

Built-in preview pane
Spotlight has this to a limited extent, but I really like the built-in preview pane for browsing through search results without remembering where they are or when they were put there
Templates
Excellent saved searches that can be easily added to or taken away from. This is the real super-feature in my mind, the opportunity to decrease repetition of previous searches.
“New Search” preferences
Similar to the above templates, it's easy to set up the preferences for what gets searched by default. (If you don't like Spotlight's default “Computer”, what are you gonna do?)

So again, it may not be terribly useful for everyone, and I believe there are already similar products out there, but the price is right (or will be, if the BlogZot is successful), so go ahead and give it a try, then write about it and register at macZOT!.

Update: Looks like this one was successful, too. If you hurry, you just might be able to head over to macZOT! and get a free registration.

Update 2: Yeah, that's over now, too. And on further look, doesn't really seem any more helpful on OS X than pressing Command-F in Finder.

Image Editing for Macs…

12/21/2005 05:06:00 PM

Why, oh why, can I not find a decent, simple image editor for OS X that does exactly what I need? I've been using an old version of Jasc Paint Shop Pro (now owned by Corel—who knew they were still around?) on Windows for years, and I love it. It's simple, and does exactly what I expect it to do. Who would have thought that I wouldn't be able to find a nice piece of artistic software for the more artistic platform?

I've tried so many different programs over the last day or two that I'm having trouble keeping them straight, so I won't go for a big “shoot-'em-out” review right now—suffice it to say that I I haven't yet found any program (free or with a free trial demo) that allows me to do the following steps:

  1. Open a Windows bitmap (BMP) image
  2. Rotate to a fine degree (say, 0.5° clockwise)
  3. Select a rectangular area
  4. Crop to that selection with a relatively simple menu selection
  5. Save the result as a new image

That's it. Want to really blow my mind? Have some simple paint functions like “line” and “text”. Really, it's all I'm looking for. I'd rather not (and won't) shell out $600 for the bloat that is PhotoShop when these are all I need. I don't suppose anyone has any suggestions? If so, please make sure it actually does what I've listed above and then please email me. Thanks!
